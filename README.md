# 🐾 GuiLab Diagnósticos

## 📷 Demonstração

![Página Inicial](images/home.png)

Sistema web desenvolvido para simular um laboratório veterinário, permitindo o acesso de diferentes perfis de usuários e a consulta de resultados de exames.

## 📖 Sobre o projeto

O **GuiLab Diagnósticos** foi criado como um projeto de estudos com o objetivo de praticar desenvolvimento web utilizando HTML, CSS e JavaScript.

O sistema possui uma interface moderna e organizada, simulando o funcionamento de um laboratório veterinário responsável pela realização e entrega de exames.

## ✨ Funcionalidades

- 🏠 Página inicial institucional
- 🔐 Sistema de login
- 👨‍⚕️ Painel do Veterinário
- 👤 Painel do Tutor
- 🛠️ Painel do Administrador
- 📄 Visualização de resultados de exames
- 📱 Interface responsiva

## ▶️ Como executar

1. Clone o repositório

```bash
git clone https://github.com/GuilhermeRibeiro12/ProjetoPessoal-Laborat-rio.git
```

2. Entre na pasta

```bash
cd ProjetoPessoal-Laborat-rio
```

3. Abra o arquivo **index.html** no navegador ou utilize a extensão **Live Server** do Visual Studio Code.
